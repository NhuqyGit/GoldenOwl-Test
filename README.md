### Run Locally
#### Golang
```bash
# 1. Move to backend directory
cd backend

# 2. Download dependencies
go mod tidy

# 3. Run the server
go run main.go
```

### Reactjs
```bash
# 1. Move to frontend directory
cd frontend

# 2. Install dependencies
npm install

# 3. Start the development server
npm run dev
```
